# CastHouse V1

CastHouse V1 MVP

## Getting started

패키지를 설치해주세요.

```bash
npm install
```

개발 서버를 기동해주세요.

```bash
npm run dev
```

[3000번](http://localhost:3000) 포트로 접속해주세요.
